# Half-Life 2: Text Edition

Короткая текстовая игра по сюжету начала Half-Life 2. Игрок делает выбор и проходит события прибытия в Сити 17.

## Запуск

1. Откройте файл `Project.exe`.
2. Следуйте инструкциям на экране.
